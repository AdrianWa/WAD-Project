About

This page was made for the Web Application Design Course at UPT.

It's backend was written in PHP using the Slim Framework. The data is stored in a SQL database. The frontend was made with HTML and CSS, the data handling is done with XMLHttpRequests, the recieved JSON fills a template using the JS framework Handlebars. To enable smooth scrolling with the Navigation Bar a the JavaScript library zenscroll is used.
